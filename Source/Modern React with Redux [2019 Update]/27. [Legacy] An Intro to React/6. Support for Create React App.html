This note is only for students who wish to use Create React App for the following projects instead of the ReduxSimpleStarter boilerplate required for the course. If you wish to use the ReduxSimpleStarter boilerplate, please continue to the next lecture.


We have created a custom script which will setup your Create React App environment so that it matches the course lectures.

Initialize the app in your terminal with this line, replacing <app_name> with your preferred project or app name.

create-react-app <app_name> --scripts-version=modern-scripts

Or, if using the new npx install method:

npx create-react-app <app_name> --scripts-version=modern-scripts


That's it! Your app will open automatically when executing an npm start at localhost:3000!
